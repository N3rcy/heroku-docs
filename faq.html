Heroku UserBot Docs - FAQ

1. I received an update in inline bot, is it worth updating??

How long ago did you stop updating just because you switched to a fork? I always write what's changed in the update, if you don't want to, don't update, live with bugs.

2. Why is there no help with beta versions?

Because beta versions are beta, there will be no help with them, because the developer knows that there are bugs or flaws and he is actively fixing and correcting them.

3. Why are some modules in /presets not downloaded?

Restart Heroku and download the preset pack again. This is due to unused dependencies.

4. When will the beta version be released?

Never. Live with it. When she's ready, she'll come out.

5. How do I get the html tag of a specific text selection? (for example, quotes)

Reply to the message with the command .e r.text, it's not difficult.

6. How do I remove the Heroku label from my info? the old way doesn't help!

Easy! Enter the fcfg herokuinfo show_heroku False command and that's it.

7. I have a no query results error, how can I fix it?

Either just restart your userbot .restart -f, or if that doesn't help, change the inline bot .ch_heroku_bot {new bot username}

8. What should I do if I get spam in logs from telegram servers?

It's simple! fcfg tester tglog_level ERROR and you will no longer be spammed. Important! this command will allow the user to send logs from telegram only with information like "error", if you are a developer, I recommend leaving it as it is.
